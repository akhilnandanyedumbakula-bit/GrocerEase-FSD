# GrocerEase (Simple Implementation)

Mini e-commerce web app for grocery items.

## Technologies
- Frontend: HTML, Bootstrap, Vanilla JS
- Backend: Node.js + Express (JSON seed data)

## Run locally
1. Clone the repo and ensure folder structure:
   - backend/
   - frontend/

2. Install backend deps:
```bash
cd task-master-completer-main
npm install

3. Running the website ;
```bash
npm run dev
